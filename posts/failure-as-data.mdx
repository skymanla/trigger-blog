# 에디슨의 전구

토머스 에디슨에게 기자가 물었습니다.
"수천 번이나 실패하셨는데, 포기하고 싶지 않았습니까?"
에디슨은 대답했습니다.
"나는 실패한 적이 없습니다. 단지 전구가 켜지지 않는 수천 가지 방법을 발견했을 뿐입니다."

## 과학자의 태도

과학자들은 실험이 실패했다고 자책하지 않습니다. 그저 "가설이 틀렸음"이라는 **데이터**를 얻었을 뿐입니다. 우리 인생도 마찬가지입니다.

- 프로젝트가 망했다면? → 이 방식은 시장에 안 먹힌다는 데이터 획득.
- 면접에 떨어졌다면? → 내 강점과 회사의 핏이 맞지 않다는 데이터 획득.
- 작심삼일에 그쳤다면? → 내 의지력이 아니라 환경 설정이 문제라는 데이터 획득.

## 감정을 빼고 팩트만 남기기

실패에 '좌절'이라는 감정을 섞지 마세요. 담백하게 팩트만 추출하세요. 그리고 다음 실험 설계(Next Step)에 반영하세요.

우리는 실패하는 것이 아닙니다.
**성공으로 가는 데이터를 수집하는 중입니다.**
